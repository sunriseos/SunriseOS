sunrise_kernel::paging::arch

[−][src]Module sunrise_kernel::paging::arch::i386

Paging implementation on i386

No PAE, no PSE, just regular 2-level paging, with simple 4kB tables and pages.

Constants

ENTRY_COUNT

The number of entries a page table has. On i386 a page table/directory is 1024 entries * 4 bytes per entry = 4kB, fits in 1 page.

PAGE_SIZE

The page size. Dictated by the MMU. In simple, elegant, sane i386 paging, a page is 4kB.
